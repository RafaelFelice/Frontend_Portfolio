# Portfólio Pessoal

![Capa do Projeto](https://i.imgur.com/G5nIJXo.png)

## 📖 Sobre o Projeto

Este é o repositório do meu portfólio pessoal, um projeto desenvolvido para apresentar minhas habilidades, experiências e projetos de forma profissional e organizada. O design foi criado no Figma e implementado com as tecnologias fundamentais do desenvolvimento front-end. O objetivo é criar uma página web responsiva, intuitiva e que reflita minha identidade como desenvolvedor(a).

🔗 **Link para o design no Figma:** [Portfólio TT no Figma](https://www.figma.com/design/oGxU2n96Qdj7pBfndHrhSa/Portf%C3%B3lio-TT)

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias:

* **HTML5:** Para a estrutura semântica e organização do conteúdo.
* **CSS3:** Para a estilização personalizada, layout e design visual.
* **JavaScript:** Para funcionalidades interativas, como o menu mobile e possíveis animações.
* **Bootstrap:** Utilizado para agilizar a criação de um layout responsivo e para componentes pré-estilizados, como o grid system.

---

## ✨ Funcionalidades

O portfólio está organizado nas seguintes seções:

* **Header (Cabeçalho):** Contém a logo e o menu de navegação principal com links para as diferentes seções da página.
* **Início (Home):** Uma seção de destaque com uma breve apresentação, sua foto e links para redes sociais como LinkedIn e GitHub.
* **Sobre Mim:** Uma descrição mais detalhada sobre minha trajetória profissional, paixões e competências.
* **Habilidades (Skills):** Apresentação das minhas principais habilidades técnicas (hard skills) e interpessoais (soft skills).
* **Projetos:** Uma galeria com os principais projetos que desenvolvi, cada um com uma breve descrição, tecnologias usadas e links para o repositório e/ou para a aplicação online.
* **Contato:** Um formulário para que visitantes possam entrar em contato diretamente pela página.
* **Rodapé:** Contém informações de copyright e links adicionais.

## 🚀 Como Rodar o Projeto

Este projeto é composto apenas por arquivos front-end (HTML, CSS, JS), então você pode visualizá-lo facilmente em seu navegador.

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/seu-usuario/seu-repositorio.git](https://github.com/seu-usuario/seu-repositorio.git)
    ```

2.  **Navegue até o diretório do projeto:**
    ```bash
    cd seu-repositorio
    ```

3.  **Abra o arquivo `index.html` no seu navegador:**
    Você pode simplesmente clicar duas vezes no arquivo `index.html` na sua pasta ou usar uma extensão como o "Live Server" no VSCode para uma melhor experiência de desenvolvimento.
